YouTube Clone (HTML & CSS)

📌 Description

This project is a YouTube homepage clone built using pure HTML and CSS. It replicates the basic layout of YouTube, including the header, sidebar, and video grid section.

The main goal of this project was learning and practicing frontend fundamentals, especially layout design and responsive styling.

🛠 Tech Stack<br>
HTML5<br>
CSS3<br>

🚀 How to Run<br>
Clone the repository<br>
git clone https://github.com/your-username/youtube-clone<br>
Open the project folder<br>
Run the project<br>
Simply open the index.html file in your browser<br>

✨ Features<br>

📺 YouTube-style video grid layout<br>
📌 Fixed header with search bar and icons<br>
📂 Sidebar navigation menu<br>
📱 Responsive design using media queries<br>
🎯 Clean and structured UI<br>
🎯 What I Learned<br>
CSS Grid for creating responsive layouts<br>
Flexbox for alignment and spacing<br>
CSS Positioning (relative, absolute, fixed)<br>
Media Queries for responsiveness<br>
Writing semantic and structured HTML<br>
