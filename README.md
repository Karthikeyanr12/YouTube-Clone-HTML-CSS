# 🎥 YouTube Clone (HTML & CSS)

## 📌 Description
This project is a **YouTube homepage clone** built using pure **HTML and CSS**. It replicates the basic layout of YouTube, including the header, sidebar, and video grid section.

The purpose of this project is to **learn and practice frontend fundamentals**, especially layout design and responsive styling.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  

---

## ✨ Features
- YouTube-style video grid layout  
- Fixed header with search bar and icons  
- Sidebar navigation menu  
- Responsive design using media queries  
- Clean and structured UI  

---

## 🎯 What I Learned
- CSS Grid for building layouts  
- Flexbox for alignment and spacing  
- CSS Positioning (relative, absolute, fixed)  
- Media Queries for responsiveness  
- Writing semantic and structured HTML


2. Open the project folder  

3. Run the project  
Open the `index.html` file in your browser
